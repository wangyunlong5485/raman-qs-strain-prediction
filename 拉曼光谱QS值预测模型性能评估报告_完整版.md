## 一、方法

### 1.1 数据集与预处理

**1.1.1 数据集描述**

本研究使用单细胞拉曼光谱数据集，共包含19,297个光谱样本，涵盖57个不同菌株。每个拉曼光谱样本包含两个通道：拉曼强度(raman intensity)和拉曼位移(raman shift)，每个通道包含2000个数据点。QS值(群体感应强度)作为连续变量进行预测，实际取值范围为[0.09, 1.38]。

数据按照QS值强度分为四类：qs强(强群体感应)、qs中(中等群体感应)、qs低(弱群体感应)和qs无(无群体感应)。这种分类确保了数据集在不同QS值区间的平衡分布，有助于模型的泛化能力评估。

**1.1.2 数据预处理**

数据预处理流程包括以下步骤：

1. 从原始.fr和.txt文件中提取拉曼光谱数据
2. 将所有光谱统一插值至2000个数据点
3. 构造双通道输入张量 $X \in \mathbb{R}^{N \times C \times L}$，其中 $N=19,297$ 为样本数量，$C=2$ 为通道数，$L=2000$ 为光谱长度
4. 采用Z-score标准化进行归一化处理：

$$
X_{\text{norm}} = \frac{X - \mu_{\text{train}}}{\sigma_{\text{train}}}
$$

$$
y_{\text{norm}} = \frac{y - \mu_{y,\text{train}}}{\sigma_{y,\text{train}}}
$$

### 1.2 模型架构: 1D ResNet

基于第一阶段结果，本研究采用ResNet作为最终模型架构。ResNet通过引入残差连接解决了深层网络的梯度消失问题：

$$
\mathcal{H}(x) = \mathcal{F}(x) + x
$$

其中 $x$ 表示输入，$\mathcal{F}(x)$ 表示残差函数，$\mathcal{H}(x)$ 表示底层映射。

**优化后的ResNet架构**:

| 组件 | 参数设置 | 说明 |
|------|----------|------|
| 初始卷积层 | Conv1d(2→64, kernel=7, stride=1, padding=3) | 提取初始特征 |
| 最大池化 | MaxPool1d(kernel=2) | 降采样 |
| 残差块数量 | 3 | 优化参数 |
| 每块滤波器 | 64×(2^(i//2)) | 指数增长 |
| Dropout率 | 0.2078 | 优化参数 |
| 全局池化 | AdaptiveAvgPool1d(1) | 全局特征聚合 |
| 全连接层 | 64→128→64→1 | 多层感知器 |

### 1.3 Optuna超参数优化

**1.3.1 优化框架**
采用Optuna贝叶斯优化框架，通过Tree-structured Parzen Estimator(TPE)算法高效搜索超参数空间。

**1.3.2 优化目标**
最小化3折交叉验证的MSE损失：

$$
\min_{\theta} \frac{1}{3} \sum_{k=1}^{3} \text{MSE}(D_k^{\text{val}}, f_{\theta}(D_k^{\text{train}}))
$$

**1.3.3 超参数搜索空间**

| 超参数 | 搜索范围 | 类型 |
|--------|----------|------|
| num_blocks | {2, 3, 4} | 整数 |
| filters_base | {32, 64, 128} | 分类 |
| kernel_size | {3, 5, 7} | 分类 |
| dropout_rate | [0.2, 0.5] | 浮点数 |
| learning_rate | [1e-5, 1e-3] (对数) | 浮点数 |
| batch_size | {128, 256, 512, 1024} | 分类 |

**1.3.4 优化过程**
- 试验次数: 50
- 并行度: 1 (串行执行)
- 早停策略: ReduceLROnPlateau + EarlyStopping
- 优化时间: 约2小时

### 1.4 训练策略

**1.4.1 损失函数与优化器**
- 损失函数: 均方误差(MSE)
- 优化器: Adam(learning_rate=0.000558)
- 学习率调度: ReduceLROnPlateau(factor=0.5, patience=10)

**1.4.2 早停策略**
监控验证集损失，连续30个epoch无改善则停止训练。

**1.4.3 训练配置**
- 最大epoch数: 200
- 批大小: 128 (4070 GPU) / 1024 (A100 GPU)
- 交叉验证折数: 10
- 随机种子: 42

### 1.5 评估方法

**1.5.1 交叉验证**
采用10折交叉验证确保结果可靠性：

$$
CV = \frac{1}{K} \sum_{k=1}^{K} \mathcal{E}(D \setminus D_k, D_k)
$$

其中 $K=10$ 为折数。

**1.5.2 评估指标**
1. **均方误差(MSE)**: $\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$
2. **均方根误差(RMSE)**: $\sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2}$
3. **平均绝对误差(MAE)**: $\frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|$
4. **决定系数(R²)**: $1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}_i)^2}{\sum_{i=1}^{n} (y_i - \bar{y})^2}$

## 三、讨论

### A. 模型加载与预测代码

```python
import torch
import numpy as np
from train_pytorch import ResNet1D, RamanDataset

def load_best_model(model_path='best_model_resnet.pth'):
    """加载最佳模型"""
    checkpoint = torch.load(model_path, map_location='cpu')
    
    # 构建模型
    model = ResNet1D(
        input_length=2000,
        params=checkpoint['params']
    )
    
    # 加载权重
    model.load_state_dict(checkpoint['model_state_dict'])
    model.eval()
    
    return model, checkpoint['scaler_y'], checkpoint['params']

def predict_qs_value(model, scaler_y, raman_spectrum):
    """预测单个拉曼光谱的QS值"""
    # 数据预处理(与训练时相同)
    # 注意: 需要实现与训练时相同的标准化逻辑
    
    with torch.no_grad():
        prediction = model(raman_spectrum)
        # 反标准化
        prediction = scaler_y.inverse_transform(
            prediction.cpu().numpy().reshape(-1, 1)
        )
    
    return float(prediction[0, 0])
```
