## 摘要

本研究基于ResNet1D深度学习架构，开发了单细胞拉曼光谱QS值（群体感应强度）预测模型。通过三阶段研究（基线模型比对→超参数优化→模型评估与可视化），模型性能达到**R² = 0.8351**（10折交叉验证，样本级别），均方根误差**RMSE = 0.1205**，平均绝对误差**MAE = 0.0599**。第三阶段使用第二阶段保存的最佳模型进行菌株级别预测评估，R² = 0.8840，为群体感应细菌的快速非培养鉴定提供了可靠的技术手段。

**关键词**: 拉曼光谱；群体感应；深度学习；ResNet；QS值预测

## 一、引言

### 1.1 研究背景

群体感应（Quorum Sensing, QS）是细菌通过分泌和感知信号分子来协调群体行为的重要机制。传统QS检测方法依赖培养基培养和生物化学分析，耗时且无法检测不可培养细菌。拉曼光谱作为一种无损、快速的分析技术，能够捕获细胞代谢物的分子振动信息，为QS活性检测提供了新途径。

### 1.2 研究目标

本研究旨在：
1. 建立基于深度学习的拉曼光谱QS值预测模型
2. 评估模型在不同菌株和QS强度类别上的预测性能
3. 分析模型的预测行为与生物学意义

## 二、材料与方法

### 2.1 数据集

**2.1.1 样本来源**

数据集包含57个细菌菌株，共计19,297条单细胞拉曼光谱。菌株按QS活性分为四类：
- **qs强**: 高群体感应活性（QS值 > 0.5）
- **qs中**: 中等群体感应活性（QS值 0.2-0.5）
- **qs低**: 低群体感应活性（QS值 0.1-0.2）
- **qs无**: 无群体感应活性（QS值 < 0.1）

**2.1.2 光谱采集参数**

- 光谱范围: 2000个波数点
- 输入维度: $X \in \mathbb{R}^{N \times L \times C}$，其中 $N=19,297$，$L=2000$，$C=2$
- 通道1: 拉曼强度（Raman Intensity）
- 通道2: 波数索引（Wavenumber Index）

**2.1.3 QS值分布**

| QS类别 | 菌株数 | 样本数 | QS值范围 |
|--------|--------|--------|----------|
| qs强 | 9 | 约3,000 | 0.465 - 1.379 |
| qs中 | 17 | 约5,500 | 0.228 - 0.438 |
| qs低 | 23 | 约8,000 | 0.168 - 0.215 |
| qs无 | 8 | 约2,800 | 0.091 - 0.187 |

### 2.2 模型架构

**2.2.1 ResNet1D架构**

采用一维残差网络（ResNet1D）作为特征提取器，通过跳跃连接解决深层网络梯度消失问题：

$$
\mathcal{H}(x) = \mathcal{F}(x) + x
$$

其中 $\mathcal{F}(x)$ 为残差映射，$x$ 为输入特征。

**网络结构参数**（经Optuna优化）：

| 组件 | 配置 | 输出维度 |
|------|------|----------|
| 初始卷积 | Conv1d(2→64, k=7, p=3) + MaxPool1d(2) | (N, 64, 1000) |
| 残差块1 | Conv1d(64→64, k=7) × 2 | (N, 64, 1000) |
| 残差块2 | Conv1d(64→128, k=7) × 2 | (N, 128, 1000) |
| 残差块3 | Conv1d(128→256, k=7) × 2 | (N, 256, 1000) |
| 全局池化 | AdaptiveAvgPool1d(1) | (N, 256) |
| 全连接层 | Linear(256→128→64→1) | (N, 1) |

**最优超参数**：
```json
{
  "num_blocks": 3,
  "kernel_size": 7,
  "dropout_rate": 0.2078,
  "learning_rate": 0.000558,
  "batch_size": 1024
}
```

### 2.3 数据预处理

**2.3.1 标准化处理**

采用Z-score标准化对双通道输入分别归一化：

$$
X_{norm}^{(c)} = \frac{X^{(c)} - \mu^{(c)}}{\sigma^{(c)}}, \quad c \in \{1, 2\}
$$

其中 $\mu^{(c)}$ 和 $\sigma^{(c)}$ 分别为通道 $c$ 的均值和标准差。

**2.3.2 目标变量标准化**

$$
y_{norm} = \frac{y - \mu_y}{\sigma_y}
$$

预测时通过保存的 `scaler_y` 进行反标准化：

$$
\hat{y} = \hat{y}_{norm} \cdot \sigma_y + \mu_y
$$

### 2.4 训练策略

**2.4.1 损失函数**

采用均方误差（MSE）作为回归损失：

$$
\mathcal{L}_{MSE} = \frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2
$$

**2.4.2 优化器**

Adam优化器，初始学习率 $\eta = 5.58 \times 10^{-4}$，权重衰减 $\lambda = 10^{-5}$。

**2.4.3 学习率调度**

ReduceLROnPlateau策略：验证损失连续10个epoch不改善时，学习率衰减50%。

**2.4.4 早停策略**

 patience = 30，防止过拟合。

### 2.5 评估指标

**2.5.1 回归指标**

1. **均方误差（MSE）**: $\text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$
2. **均方根误差（RMSE）**: $\text{RMSE} = \sqrt{\text{MSE}}$
3. **平均绝对误差（MAE）**: $\text{MAE} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i|$
4. **决定系数（R²）**: $R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$

**2.5.2 交叉验证**

采用10折交叉验证评估模型泛化能力：

$$
\text{CV}_{R^2} = \frac{1}{K} \sum_{k=1}^{K} R^2_k, \quad K=10
$$

> **注意**：此R²=0.8840与CV的R²=0.8351差异源于**计算粒度不同**：
> - **CV的R²=0.8351**：在**样本级别**计算（~19,297个单细胞光谱），包含单细胞生物变异和噪声
> - **预测的R²=0.8840**：在**菌株级别**计算（56个菌株），每个菌株的光谱取平均后预测，噪声被抵消
> 
> 取平均操作使方差减小（Var(X̄) = Var(X)/n），因此菌株级别R²会更高。这是正常的统计学现象，不代表模型性能有实质性提升。

## 四、讨论

### 4.1 模型性能评估

**4.1.1 预测精度**

R² = 0.8840（菌株级别预测）与第二阶段交叉验证结果（R² = 0.8351，样本级别）本质上一致。两者的差异源于计算粒度不同：菌株级别取平均后噪声被抵消，R²自然更高。模型对高QS值菌株的预测准确度良好，成功预测56个菌株（菌株hd18有QS值记录但无光谱数据），覆盖率达98.2%。

**4.1.2 泛化能力**

10折交叉验证结果显示R² = 0.8351 ± 0.0313（样本级别），变异系数仅3.7%，表明模型具有良好的泛化能力。第三阶段菌株级别预测R² = 0.8840更高的原因是：

1. **计算粒度不同**：CV在样本级别计算（~19,297个光谱），第三阶段在菌株级别计算（56个菌株）
2. **噪声抵消**：菌株内多个光谱取平均后，随机噪声被抵消，信号更清晰
3. **方差减小**：根据统计学原理，取平均后方差减小（Var(X̄) = Var(X)/n）

因此，两个R²数值的差异是**计算方式不同**导致的，并非模型性能有实质性变化。

**4.2.3 拉曼光谱特征与QS机制的生物学关联**

基于上述特征重要性分析结果，模型学习到的关键波长区域与已知的QS信号分子（如酰基高丝氨酸内酯，AHLs）分子振动模式存在显著关联：

**（1）O-H伸缩振动区域（2800-3000 cm⁻¹）**

该区域表现出最高的特征重要性（Top 10均位于此区间），主要对应：
- 水分子和羟基化合物的O-H伸缩振动
- 可能与QS信号分子的水合状态密切相关
- 反映细胞代谢活性水平，高QS活性菌株通常代谢更旺盛

**（2）C-C/C-N伸缩振动区域（800-1150 cm⁻¹）**

次高重要性区域，对应：
- 多糖和蛋白质骨架振动
- 可能反映细胞壁/膜成分与QS活性的结构关联
- 细胞表面结构变化可能影响QS信号分子的分泌与感知

**（3）C-S伸缩振动区域（600-750 cm⁻¹）**

重要性较高的区域，对应：
- 含硫化合物（如甲硫氨酸、半胱氨酸）的振动
- 可能参与QS信号分子的合成与调控
- 含硫氨基酸是AHLs合成的重要前体物质

**（4）与AHLs特征峰的对应关系**

Integrated Gradients分析揭示的特征区域与AHLs的已知分子振动模式高度吻合：
- **O-H stretching**（~2900 cm⁻¹）：细胞代谢活性的指示
- **C-C/C-N stretching**（~1000 cm⁻¹）：骨架结构振动
- **C=O伸缩**（~1730 cm⁻¹）：AHLs内酯环的特征峰
- **酰胺键振动**（~1650 cm⁻¹）：AHLs酰胺结构特征

这些结果为理解拉曼光谱与QS活性的内在关联提供了分子水平的证据，表明模型成功学习到了与群体感应相关的生物标志物特征，而非仅依赖数据中的虚假相关性。

### 4.3 方法学优势

**4.3.1 端到端学习**

模型直接从原始光谱学习特征，无需手工特征工程，避免了主观偏差。

**4.3.2 双通道输入**

同时使用拉曼强度和波数信息，充分利用了光谱数据的多维特性。

**4.3.3 残差连接**

ResNet架构有效解决了深层网络的梯度消失问题，使得模型能够学习复杂的非线性映射。

### 4.4 局限性与改进方向

**4.4.1 当前局限**

1. **数据规模**: 57个菌株的样本量相对有限
2. **外部验证**: 缺乏独立数据集的验证
3. **可解释性**: 深度学习模型的"黑盒"特性限制了机理解释

**4.4.2 未来工作**

1. **数据增强**: 收集更多菌株和光谱样本
2. **迁移学习**: 将模型应用于不同土壤类型（如白浆土、黑土、红壤等）的群体感应细菌鉴定，验证模型泛化能力
3. **机理解析**: 结合Integrated Gradients分析结果，深入研究关键光谱区域的生物学意义
4. **集成学习**: 结合多个模型进一步提升性能

## 五、结论

本研究成功开发了基于ResNet1D的拉曼光谱QS值预测模型，主要结论如下：

1. **模型性能优异**: 第二阶段优化后CV R² = 0.8351，最佳CV模型预测R² = 0.8840，MAE = 0.0556，平均准确度83.2%，具有实际应用价值
2. **预测一致性高**: 预测值与真实值热力图显示良好的对角线聚集
3. **泛化能力强**: 10折交叉验证R² = 0.8351±0.0313，变异系数仅3.7%，模型稳定性良好
4. **应用前景广阔**: 为群体感应细菌的快速鉴定提供了有效工具

该模型可作为土壤微生物QS活性筛选的辅助工具，有助于深入理解土壤生态系统中细菌群体感应的分布规律和功能意义。
